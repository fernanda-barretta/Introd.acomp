Me ajuda! é um gerador de plano de estudos desenvolvido por Fernanda Santos e Natalia Oliveira, pensado para estudantes que têm dificuldade de se organizar ou que acabam procrastinando demais na hora de estudar. A partir de algumas perguntas simples — como o nível de dificuldade da matéria, o conhecimento prévio e os dias disponíveis até a avaliação — o site gera um cronograma personalizado para ajudar o estudante a chegar bem preparado. O projeto foi desenvolvido com HTML e CSS utilizando o framework Tailwind CSS, como parte das aulas de Desenvolvimento Web.
